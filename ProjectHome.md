## Natív vizuális élményt nyújtó vékonykliens megoldások Mozilla környezetben ##

> A szoftverfejlesztési iparág igen nagy szeletét adják a különböző vállalatok, cégek, szervezetek, közigazgatási szervek, intézmények belső működését, irányítását célzó háttérrendszerek. A napi munkavégzés során gyakran bizonyul elégtelennek a hagyományos HTML-en és HTTP-n alapuló megoldások.

> A Mozilla alkalmazások alapját egy XUL alapú platform adja, amely egy XML-re és JavaScript-re épülő megoldást nyújt akár WEB-es alkalmazások futtatásához is. Az így kialakított szoftverek WEB-es vékony kliens technikával is natív kliens oldali alkalmazás élményét nyújthatják.

> A feladat PHP API kialakítása, melynek segítségével hatékonyan és könnyen lehet szerver oldali implementációval Mozilla Platformon futó alkalmazásokat fejleszteni. Az API megvalósítására a Yii PHP nyelvű web-fejlesztési keretrendszer használandó. A kialakítandó API működőképességét egy példa alkalmazáson keresztül is be kell mutatni.
